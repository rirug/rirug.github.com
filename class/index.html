RIRUG Ruby Install Night at Bryant University

Getting started

  • Get on the wi-fi. You'll need it.
  • Sign up for a free CodeSchool account. This account will let you access some excellent online tutorials that we will use during this event.
  • Get Ruby and Rails installed on your laptop.
    • Linux: Install RVM
      "\curl -L https://get.rvm.io | bash -s stable --rails --autolibs=enabled"
    • Windows: Use RailsInstaller (also available on DVDs)
    • OSX: Install XCode, then install RVM or RailsInstaller
